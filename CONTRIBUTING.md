# Contributing

Thanks for your interest in RTVox.

## Scope

This repository is currently focused on:

- open codec experiments
- bitstream design
- cross-language compatibility
- low-bitrate voice coding ideas
- reproducible reference implementations

## Rules for contributions

1. Keep the bitstream documented.
2. Do not change the container format silently.
3. Maintain compatibility tests across C, Python, and Rust.
4. Add or update documentation when behavior changes.
5. Keep the reference implementation portable.
